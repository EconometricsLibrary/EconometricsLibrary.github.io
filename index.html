Econometrics Free Library

A collection of free (GNU-FDL) Econometrics textbooks

View My GitHub Profile

The goal of this project is to produce and maintain up-to-date Econometrics texts suitable for graduate education and available for free under the GNU Free Documentation License.

Texts under development (pre-alpha)

  • Core Econometrics Text, disorganized notes for a first-semester PhD econometrics core class.
  • Graduate Macroeconometrics, at the moment, a disorganized set of partial lecture notes for a PhD time-series elective.

Support or Contact

We have a mailing list: econometricslibrary@librelist.com. Please email the list if you have questions or error reports; the Librelist homepage has more information about this process; list archives are available online as well. You can also file issues for each of the individual books:

  • Core Econometrics text issues
  • Graduate Macroeconometrics text issues

Please subscribe to the mailing list if you'd like to contribute to this project. I plan to eventually add an IRC channel, but we're not large enough to justify it yet.
