Econometrics Free Library

A collection of free & open access (GNU-FDL) Econometrics textbooks

The goal of this project is to produce and maintain up-to-date Econometrics texts suitable for graduate education and available for free under the GNU Free Documentation License.

Texts under development (pre-alpha)

  • Core Econometrics, disorganized notes for a first-semester PhD econometrics core class (downloads).

Support or Contact

We have a mailing list: econometricslibrary@librelist.com. Please email the list if you have questions or error reports; the Librelist homepage has more information about this process; list archives are available online as well. You can also file issues when you discover errors:

  • Core Econometrics text issues

Please subscribe to the mailing list if you'd like to contribute to this project.

There is also a (mostly empty) general wiki for the entire project, as well as individual wikis for each document.

Authors and Contributors

This project was started on April 29, 2013 by @grayclhn; you can find more information on GitHub: @EconometricsLibrary. Each book lists its contributors/authors separately.
